# Catalog Infinite Scroll (TheCodingTutor_Infinitescroll) v1.0.0
## Magento 2 Extension

Catalog Infinite Scroll provides infinite ajax scroll that auto load the next page. Offer user friendly UI, help customer save clicks & time.

## Features:
1. Magento 2 Ajax scroll without interruption
2. Can restrict infinite scroll to particular screen width
3. Place navigation bar in any location at the page.
4. Similar technique as seen on Twitter, Facebook
5. Used in category page and search page
6. Easy to use
